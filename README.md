# 🚀 InsureYouAI
**AI-Powered Insurance Management & Intelligence System**

InsureYouAI is a sophisticated backend project developed using **ASP.NET Core** and **MSSQL**. It integrates multiple AI models to provide automated content management, sentiment analysis, user moderation, image generation, and real-time AI-driven chatbot capabilities.

The project is designed around a corporate insurance scenario, merging traditional enterprise backend processes with cutting-edge AI integrations.

---

## 🎯 Project Objectives

- **Automated Moderation:** Leveraging AI to manage content and user interactions in insurance systems.
- **Sentiment & Behavioral Analysis:** Analyzing user comments and behaviors for automatic classification and risk detection.
- **Real-time Interaction:** Providing a seamless experience with an AI-powered chatbot.
- **Multi-Model Integration:** Integrating various AI services into a single unified backend.
- **Architectural Excellence:** Building a sustainable and extensible system following SOLID principles and Clean Code.

---

## 🧠 AI Ecosystem

This project stands out by its ability to orchestrate multiple AI providers, ensuring zero dependency on a single vendor:

- **LLMs:** OpenAI (GPT), Google Gemini, Anthropic Claude, Microsoft Azure AI.
- **Creative AI:** Stability AI & Replicate for dynamic image generation.

---

## ⚙️ Key Features

### 🔹 Intelligent Content Management
- Automatic content generation for blogs and announcements using AI.
- Dynamic management of text-based corporate assets.

### 🔹 Sentiment Analysis & Moderation
- Detection of toxic, inappropriate, or fraudulent content.
- Automated classification based on user feedback.

### 🔹 User Behavior Profiling
- Generating user profiles based on interaction history.
- Identification of high-risk users through behavioral analysis.

### 🔹 AI-Driven Image Generation
- Generating custom visuals for policies or marketing materials via AI.
- Full integration of generated assets into the system workflow.

### 🔹 Real-time AI Chatbot
- **SignalR Integration:** High-performance real-time messaging.
- **Streaming Responses:** Smooth, real-time AI response streaming for an enhanced user experience.
